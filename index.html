This is a demo of an elastic website a little bit more complex than what is described in How to develop an elastic website.

Pretty normal... Until you go to Preferences, Content, Fonts & Colors (in Firefox) and change the font size: if the font size chosen is too big the page becomes a stack of elements, just like on small mobile devices. No broken layout.

This was written in a rush so I'm pretty sure there's a lot of room for improvement. I think I can reduce the media queries dependant code.
